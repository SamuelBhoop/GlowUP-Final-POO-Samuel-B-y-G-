# GlowUp (versión consola)

Requisitos: Java 17+.

## Ejecutar en IntelliJ
1. Abra IntelliJ → New Project (Java) → Finish.
2. Copie la carpeta `src` de este proyecto dentro del proyecto de IntelliJ (o márquela como *Sources Root*).
3. Ejecute `app.Main` (botón verde).

Usuarios de prueba:
- Dueña: `sakura@glowup.com` / `1234`
- AdminContenido: `editor@glowup.com` / `1234`

Flujo: Registrar cliente → Log in (cliente) → Listar productos → Agregar al carrito → Registrar compra.


## Paquetes adicionales
- `produccion`: `Fabrica`
- `operaciones`: `TrabajadorEsclavizado`, `RegistroEsclavos`, `ConsejoSombrio`
